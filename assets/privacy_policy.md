# PaperHealth 隐私政策

**生效日期：2026年1月5日**

欢迎使用 PaperHealth (PHF)。我们深知个人医疗信息的敏感性，因此在设计本应用时，将隐私与安全作为核心宪章。

## 1. 核心承诺：100% 本地化
PaperHealth 是一款**完全离线**运行的应用。
- **无网络请求**：应用核心功能不需要任何网络连接，应用也不会向任何外部服务器发送您的数据。
- **本地存储**：您的所有病历记录、照片、OCR 识别文本及标签均仅存储在您的设备本地。
- **本地处理**：所有的计算任务（如图片压缩、OCR 文字识别）均利用您设备的自身算力完成。

## 2. 数据加密
为了保护您的数据免受未授权访问，我们采用了以下安全措施：
- **数据库加密**：使用 SQLCipher 算法对本地数据库进行整库加密。
- **文件加密**：所有的图片附件在存入应用私有目录前，均使用 AES-256 算法进行独立加密。
- **安全锁**：应用提供 PIN 码及生物识别（指纹/面容）安全锁，作为访问数据的第一道防线。

## 3. 数据控制权
您拥有对数据的绝对控制权：
- **手动备份**：应用提供加密备份功能，备份文件（.phbak）由您自行保管。
- **物理删除**：当您在应用内删除记录时，我们会执行物理擦除逻辑，确保数据从磁盘中彻底消失。
- **应用卸载**：若您卸载应用，系统沙盒内的所有加密数据及密钥将随之被系统彻底清除。

## 4. 权限使用说明
- **相机**：用于拍摄纸质病历资料。
- **相册**：用于导入已有的病历图片。
- **生物识别**：用于应用启动时的身份验证。

## 5. 联系我们
由于本应用不收集任何数据，我们无法通过应用主动联系您。如果您有任何疑问或反馈，请通过应用商店提供的联系方式或开源社区与我们联系。
